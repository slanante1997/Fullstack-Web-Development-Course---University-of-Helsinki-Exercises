# University of Helsinki - Fullstack Course Exercise 1.1 - 1.5: Components and Props

This repository contains my solution to [Exercise 1.1 - 1.5](https://fullstackopen.com/en/part1/introduction_to_react#exercises-1-1-1-2) from the University of Helsinki's Fullstack Course. In this exercise, we focus on understanding components and props in React, which are fundamental concepts for building reusable and modular user interfaces.

## Exercise 1.1 - 1.5 Objectives

The main objectives of this exercise are to:

1. Develop a basic understanding of React components and how they can be used to create reusable UI elements.
2. Learn how to pass data between components using props.
3. Implement a simple React application that demonstrates the usage of components and props.
